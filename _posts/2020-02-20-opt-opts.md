How do we find a provably optimal set of options?
We consider a meta-RL setting, where we want to find an optimal set of options that are useful for a distribution of tasks.
Finding an provably optimal set of options may be more difficult than finding an optimal policy in an MDP.

One hope is that we can iteratively greedily select options to add to an existing set of options.
However, the objective is not submodular so we may not have an optimality guanrantee.
